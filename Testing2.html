Lesson 1: Java Syntax

Like all programming languages, Java has its own syntax, and the definition of syntax is basically how the code is supposed to be written and structured.

For example, down below you will see some code that has the words "class", "main", and "int". These words are representing containers in the code that each contain some different thing or VALUE.

For coding, it really just comes down to what contains what and which things can contain what values and for what reason. Then after you code structuring down, all you have to worry about is thinking about what function your code is going to bring to your program, and what you want your program to do and the logic behind doing it.

public static void main(String[] args) {

int integer = 100;

System.out.println("Number: " + integer);
